Performance Survivors

Performance Survivors is a test to see what performance the Godot Servers are capable of providing.


Made a performance test using #godot4, drawing 1792 objects with collision detection on a 7-year-old laptop at 135 fps.

Of course this is not possible using nodes, I am using servers and the object pooling technique.
